(7) How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?

Ans. Installation of react js windows 1)Install Node.js installer for Windows.
2)Check Node js and npm version
node -v npm -v
3)Now in the terminal run the below command: 1) npx create-react-app app-name 2) cd app-name 3) npm start
Installation of react js linux
1)Install Node js and Npm with below commands sudo apt update sudo apt install nodejs sudo apt install npm
2)then install rect with below command npm install -g create-react-app
3)and create new react application create-react-app my-app cd my-app npm start
